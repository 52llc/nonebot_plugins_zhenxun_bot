# nonebot-plugin-petpet

安装相关依赖

```
pip3 install -r requirement.txt
```

### 使用

- 指令 + @user，如： 爬 @小Q
- 指令 + qq号，如：爬 123456
- 指令 + 自己，如：爬 自己
- 指令 + 图片，如：爬 [图片]
