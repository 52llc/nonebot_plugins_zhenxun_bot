# 头像相关表情生成

来一起愉快的恶搞别人的头像吧！
需要安装的依赖

```
pip install imageio
```

    指令:
        摸/搓/撕/丢/爬/亲/贴/精神支柱 {qq/@user/自己/图片}
    参数解释:
        qq: qq号
        @user: @某个人
        自己: 生成自己的图片
        图片: 自定义图片（不是方形会被非等比缩放）
